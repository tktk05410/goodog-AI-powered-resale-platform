## [v2.1.0] - 2026-06-16

### 🛡️ 管理员后台系统

- **新增管理员角色**：`user` / `admin` 双角色体系
- **管理员账号**：用户名 `admin`，密码 `admin12345`
- **管理后台功能**：
  - 数据概览：注册用户、商品总数、在售/已售、消息总数
  - 用户管理：查看所有用户，支持删除非管理员账号
  - 商品管理：查看所有商品，可直接编辑或删除任意商品
  - 系统日志：查看系统操作记录
  - 消息中心：保留普通用户的聊天功能
- **权限控制**：
  - 管理员可编辑/删除任意商品（不受用户 ID 限制）
  - 导航栏根据角色显示"管理"或"消息"入口

### 📝 求购功能优化

- **求购商品新增"理想价格"字段**：发布和编辑求购商品时可填写期望价格
- **求购商品详情页按钮优化**：
  - 出售商品："立即购买" + "联系卖家"
  - 求购商品："我有这个商品" + "联系买家"
- 移除求购商品的支付流程，避免误操作

### 🎨 UI 交互优化

- **主页合并**：删除独立 Home.vue，`/` 默认重定向到商品列表页
- **导航栏激活样式**：当前页面标签显示黑色圆角背景 + 白色文字
- **退出登录按钮**：从导航栏移至个人中心"修改密码"旁
- **统计页面新增导航栏**：与商品/管理/统计页统一
- **管理后台布局**：左右区域宽度与导航栏内容区对齐
- **编辑页面 UI 统一**：更新为 NOMAD 极简黑白风格

### 🤖 AI 功能增强

- **AI 文案生成**：发布商品时可一键生成商品描述文案
- 修复 AI 文案功能超时问题，恢复原始版本

### 📊 数据统计修复

- 修复用户活跃时段 UTC 时间转换异常（东八区 +8 小时）

### 🧪 示例数据完善

- 创建 5 个测试账号（小明同学、学姐 Laura、数码达人、考研小张、健身阿凯）
- 添加跨账号示例对话，模拟真实使用痕迹（不同时段分布）
- 上架 12 个示例二手商品，覆盖手机、显卡、图书、健身等类别
- 每个商品配置对应图片和标签

## [v2.0.0] - 2026-06-14

### 🎨 UI风格全面升级 - NOMAD极简黑白设计系统

#### 视觉设计重构
- **采用NOMAD品牌极简黑白设计语言**
- 全局配色方案：
  - 主色：纯黑 `#000000`
  - 背景：白色 `#FFFFFF` + 浅灰页面底色 `#F8F8F8`
  - 文字层级：深黑 `#1A1A1A` / 灰色 `#666666` / 浅灰 `#999999`
  - 边框：淡灰 `#E5E5E5`（营造轻盈感）
  
- **字体系统升级**：
  - Logo/大标题：28px, 字重800 (Extrabold), 负字间距
  - 页面标题：36px, 字重700 (Bold)
  - 章节标题：20px, 字重600 (Semibold)
  - 正文：15px, 字重400 (Regular)
  - 小字说明：11-13px, 字重500 (Medium)

- **间距与留白优化**：
  - 页面边距：48px（原20px）
  - 模块间距：48px（原20px）
  - 卡片内边距：32px（原20px）
  - 元素间距：12-24px（原8-16px）

- **组件样式统一**：
  - 圆角：10px/14px/20px三级圆角
  - 阴影：极轻阴影 `0 2px 8px rgba(0,0,0,0.08)`
  - 按钮：黑色实心、白色文字、hover上浮+阴影
  - 卡片：白色背景 + 淡边框 + hover放大效果
  - 导航栏：毛玻璃效果 `backdrop-filter: blur(10px)`

#### 📄 页面样式改造（8个页面）

1. **首页 (Home.vue)**
   - 导航栏毛玻璃效果 + 黑色Logo
   - 搜索栏独立圆角矩形（四边圆润20px）
   - 筛选器 + 发布商品按钮同行布局
   - 商品卡片hover动画（上浮6px + 图片缩放1.05x）
   - 导航链接下划线动画效果

2. **商品列表页 (Products.vue)**
   - 卡片hover动画优化
   - 图片微缩放过渡效果
   - 筛选器样式统一

3. **商品详情页 (ProductDetail.vue)**
   - 左右分栏布局（图片区 + 信息区）
   - 超大价格展示（48px Extrabold）
   - 支付对话框美化

4. **个人中心页 (Profile.vue)**
   - 统计数据超大数字展示（36px）
   - 卡片式信息展示
   - 头像圆形设计（96px）

5. **消息页面 (Messages.vue)**
   - 聊天气泡黑白配色
   - 自己消息：黑色实心气泡
   - 对方消息：灰色背景气泡
   - 会话列表hover高亮效果

6. **统计页面 (Stats.vue)**
   - 概览卡片网格布局
   - 数据数字超大展示（36px Extrabold）
   - 图表容器美化

7. **登录/注册页面 (Login.vue / Register.vue)**
   - 去除渐变背景（原蓝紫渐变）
   - 纯白卡片 + 大阴影
   - 黑色按钮风格

8. **发布商品页 (Publish.vue)**
   - 表单元素圆角统一
   - 标签输入区域优化
   - AI文案按钮样式调整

#### 🔧 Element Plus组件全局覆盖

- **标签 (el-tag)**:
  - 默认标签：浅灰底 `#F5F5F5` + 黑字
  - 警告标签（求购）：浅黄底 `#FFF9E6` + 深黄字 `#B8860B`
  - 危险标签（已售出）：浅红底 `#FFF1F0` + 红字 `#CF1322`

- **Tab标签页 (el-tabs)**:
  - 激活态：黑色下划线（3px高度）+ 粗体文字
  - Hover状态：颜色渐变过渡

- **按钮 (el-button)**:
  - 主按钮：黑色背景 + 白色文字 + hover上浮
  - 默认按钮：透明背景 + 黑色边框 + hover边框加深

- **表单元素**:
  - 输入框：大圆角(10px) + focus时黑色边框 + 阴影光晕
  - 文本域：统一样式
  - 单选按钮组：黑色选中态

- **其他组件**:
  - 徽章(el-badge)：黑色背景
  - 复选框/开关：黑色激活色
  - 对话框：大圆角14px
  - 分页器：黑色激活态

#### 📐 布局结构优化

- 最大宽度扩展至 `1320px`（原1200px）
- 导航栏高度 `72px`
- 商品卡片最小宽度 `280px`（原240px）
- 商品详情图片高度 `500px`（原400px）
- 列表页图片高度 `240px`（原180px）
- 响应式Grid自适应布局

#### 💻 技术实现细节

- **CSS变量系统**（Design Tokens）：
  - 颜色变量：12个
  - 字体变量：10个
  - 间距变量：8个
  - 圆角变量：4个
  - 阴影变量：4个
  - 布局变量：3个
  
- **动画系统**：
  - 所有过渡使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动函数
  - Hover状态统一添加 `transform: translateY(-6px)` 和 `box-shadow: lg`
  - 图片hover缩放：`transform: scale(1.05)`
  - 过渡时长：0.25s-0.45s

- **新增文件**：
  - `项目结构设计.md`：项目架构文档

## 版本说明

### 版本号规则
- **主版本号 (MAJOR)**：不兼容的API修改或重大架构变更
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正

### 更新类型标识
- `🎨` - UI/视觉改进
- `✨` - 新功能
- `🔧` - 技术改进
- `📄` - 文档更新
- `🔒` - 安全修复
- `⚡` - 性能优化
- `🐛` - Bug修复
- `📊` - 统计数据
